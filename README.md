# Yesnopuzzle

Demo Ember.js repository for my presentation on Devclub.lv. 

## Versions
* Ember.js 1.8.0
* Ember Data 1.0.0-beta.11  

## Installation

* `npm install`
* `bower install`

## Running / Development

* `ember server`
* Visit your app at http://localhost:4200.
